## 📱 Responsive Design

The portfolio is fully responsive and optimized for:
- Desktop (1920px and above)
- Laptop (1024px - 1919px)
- Tablet (768px - 1023px)
- Mobile (320px - 767px)

## 🎨 Customization

To customize the portfolio:

1. Update content in `index.html`
2. Modify colors in CSS variables (`:root` section)
3. Replace placeholder images with your actual work
4. Update links and contact information
